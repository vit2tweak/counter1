## Getting Started
1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open http://localhost:3000 in your browser

## Dependencies
- React 18.2.0
- Material-UI 5.x
- Material Icons 5.x
